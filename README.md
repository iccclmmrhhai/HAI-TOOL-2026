OverviewThe HAI Deciphering Suite is a lightweight, browser-based clinical decision support tool customized for the Corazon Locsin Montelibano Regional Hospital (CLMMRH). It is designed to assist Infection Preventionists and Healthcare Providers in accurately classifying Healthcare-Associated Infections (HAIs) based on the 2026 CDC NHSN (National Healthcare Safety Network) definitions.Core FeaturesThree Integrated Modules:CLABSI: Logic for LCBI 1 (Recognized Pathogen) and LCBI 2 (Common Commensal with matching cultures/symptoms).CAUTI: Logic for SUTI 1 based on culture thresholds ($≥10^5$ CFU/ml) and clinical symptoms.SSI: Automated 30-day and 90-day surveillance window tracking based on procedure type.The "2-Day Rule" Engine: Automatically differentiates between Present on Admission (POA) and HAI based on the Hospital Day of Event.Electronic Case Log: Saves up to 10 recent analyses locally on the device.Data Portability: Feature to export the case history into a CSV file for use in Excel/reporting.Built-in Manual: Instant access to CDC definitions and user instructions within the app.Installation InstructionsNo installation or internet connection is required once the file is saved.Open a plain text editor (Notepad on Windows or TextEdit on Mac).Paste the provided HTML code into the editor.Save the file as CLMMRH_HAI_Tool.html.Launch: Double-click the file to open it in any modern web browser (Chrome, Edge, Safari, or Firefox).How to UseSelect Module: Choose BSI, UTI, or SSI from the dropdown.Input Data: Enter the Hospital Day (Day 1 = Admission Day) and specific lab/clinical findings.Analyze: Click "Analyze & Save Case."Review Logic: The result box will display the determination (POSITIVE/NEGATIVE) and the specific NHSN rationale applied.Export: At the end of your shift, go to the Case Log tab and click Download Log to save your data.Technical RationaleThe app utilizes a deterministic logic tree to eliminate human error in calculating windows.Privacy & SecurityLocal Storage: All patient identifiers (Room/Initials) are stored only in the browser's local cache on your specific device.Zero Data Transmission: No data is sent to the cloud or external servers, ensuring HIPAA/Data Privacy compliance as long as the device itself is secured.MaintenanceThis tool is based on the 2026 NHSN Manual. If the CDC updates definitions in 2027, the logic within the <script> tag of the HTML file should be updated accordingly.
